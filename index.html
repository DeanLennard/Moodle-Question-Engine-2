Moodle-Question-Engine-2 by timhunt

This is work-in-progress of my rewrite of the Moodle question engine.

Work is progressing on the new_qe branch, which is the default branch. This is a branch off the MOODLE_19_STABLE branch.

To see the new code in action, go into the question bank and create questions (that has not changed) and then preview the questions in the question preview pop-up window. You can also run the unit tests by going to Administration -> Reports -> Unit tests. Choose the option 'Only run tests in' 'question'.

The new code is in question/engine and question/interaction. There are also interesting changes in question/type.

Note that new_qe branch gets rebased regularly, so you cannot just follow developments using git pull. You will get errors about the merge not being a fast-forward if a rebase has been done. When that happens, the easiest solution is to delete your old copy of the new_qe branch, and checkout a new one.

Install

  1. Checkout the new_qe branch.
  2. Install Moodle as usual.
